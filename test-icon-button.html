Icon Button Test

Open the browser console and navigate to http://localhost:5173

Test Code to Try:

// Test 1: Icon only button
<Button icon={<PencilIcon title="Rediger" />} />

// Test 2: Icon with text
<Button icon={<PencilIcon aria-hidden />}>Rediger</Button>

// Test 3: Complete example with variant
<Button variant="primary" icon={<PencilIcon aria-hidden />}>Edit</Button>

Instructions:

  1. Open http://localhost:5173 in your browser
  2. Paste one of the test codes above into the editor
  3. Check that the preview renders the button with the icon
  4. Open the browser console and verify:
    • ✅ Sandbox ready message shows icons loaded
    • ✅ window.PencilIcon type should be "function"
    • ✅ No errors about PencilIcon not being defined

Debug Commands (in browser console):
