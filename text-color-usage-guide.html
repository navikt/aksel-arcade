How Text Colors Work in Aksel Darkside

Text colors are defined by TWO props working together:

Prop Purpose Values
textColor Controls the intensity/variant default, subtle, decoration, contrast
data-color Controls the color type neutral, accent, success, warning, danger, info, brand-magenta, brand-beige, brand-blue, meta-lime, meta-purple

Example 1: Accent Color with Subtle Intensity

<Heading 
  level="1" 
  textColor="subtle" 
  data-color="accent"
>
  Accent Subtle Heading
</Heading>

This renders text with --ax-text-accent-subtle color

Example 2: Brand Magenta with Default Intensity

<BodyLong 
  textColor="default" 
  data-color="brand-magenta"
>
  Brand magenta body text
</BodyLong>

This renders text with --ax-text-brand-magenta color

All Available Combinations

You can combine any textColor intensity with any data-color type:

  • textColor="default" data-color="accent" → --ax-text-accent
  • textColor="subtle" data-color="accent" → --ax-text-accent-subtle
  • textColor="decoration" data-color="accent" → --ax-text-accent-decoration
  • textColor="contrast" data-color="accent" → --ax-text-accent-contrast
