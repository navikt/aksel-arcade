Expected Results

  • ✅ textColor prop appears in autocomplete for Heading, BodyLong, BodyShort, Label, Detail, ErrorMessage
  • ✅ When typing textColor=", autocomplete shows all 44 Darkside text color tokens
  • ✅ Text colors actually change in the preview pane
  • ❌ If colors don't change, the issue is likely that Aksel doesn't support the textColor prop with CSS variable values

Debugging Steps

  1. Open browser DevTools Console
  2. Check for any errors related to textColor prop
  3. Inspect the rendered HTML element
  4. Check if the textColor prop is being applied as an attribute or style
  5. Verify the computed styles show the correct color value
